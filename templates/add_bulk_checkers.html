Upload a ZIP file containing the checkers.
ZIP File Structure Requirements:

The ZIP file must contain a directory for each checker. Each checker directory must include:

  • A Python script (e.g., checker.py).
  • A metadata.json file with the checker's name, expected flag, and optional environment variables.

Example structure:

checkers.zip/
├── checker1/
│   ├── checker.py
│   └── metadata.json
└── checker2/
    ├── checker.py
    └── metadata.json
